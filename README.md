# Ідентифікація бізнес-процесів

## Опис системи

Обрана система: **Інтернет-магазин**.  
Інтернет-магазин продає різноманітні товари онлайн і забезпечує покупцям зручний спосіб купівлі з доставкою додому. Користувачами системи є покупці, які реєструються на сайті, та менеджери магазину, які обробляють замовлення. Цінність системи полягає в економії часу покупців та зручності замовлення товарів з доставкою додому. Магазин надає такі послуги: оформлення замовлення, обробка оплати, доставка товару, повернення товару та технічна підтримка клієнтів.

## Цінність процесів

| Процес                | Цінність                                      | Без процесу                             |
|-----------------------|-----------------------------------------------|----------------------------------------|
| Реєстрація користувача | Дозволяє створити акаунт і відслідковувати замовлення | Користувач не може купувати          |
| Оформлення замовлення  | Забезпечує покупку товару                     | Продаж неможливий                       |
| Обробка оплати         | Забезпечує завершення покупки                 | Продаж неможливий                       |
| Доставка товару        | Доставляє товар клієнту                       | Покупець не отримає товар              |
| Управління складом     | Підтримує наявність товару та контроль запасів| Замовлення не можна виконати           |
| Обробка повернень      | Підтримує клієнтську довіру                   | Клієнти незадоволені                     |
| Підтримка клієнтів     | Допомагає вирішувати проблеми                 | Клієнти залишаються без допомоги       |
| Аналітика продажів     | Дає змогу приймати управлінські рішення       | Бізнес не розвивається                  |
| Управління каталогом товарів | Забезпечує актуальність товарів           | Клієнти бачать неправильний асортимент |
| Маркетингові розсилки  | Залучає нових клієнтів                         | Менше продажів                          |

## Висновок

Ідентифікація бізнес-процесів дозволяє зрозуміти, як працює інтернет-магазин, які процеси створюють цінність для клієнтів, а які підтримують або управляють системою. Використання Markdown-таблиць у README.md дозволяє наглядно представити класифікацію процесів та їх цінність, а ключові процеси для автоматизації допомагають оптимізувати роботу системи та підвищити ефективність.
